Explicación del código:
1.	HTML Structure:
o	Un div con la clase video-selector-container envuelve todo para permitir el centrado y aplicar un estilo de tarjeta.
o	Un h1 para el título "Seleccione un vídeo:".
o	Un div con la clase select-wrapper envuelve al elemento <select>. Esto es crucial para poder añadir la flecha personalizada usando CSS ::after sin que el navegador la superponga con su propia flecha.
o	El <select id="videoSelect"> contiene las opciones.
	La primera option tiene value="" disabled selected para que sea la opción por defecto y no sea seleccionable como un valor real.
	Las siguientes opciones tienen los nombres de archivo de vídeo.
2.	CSS Styling:
o	Centrado Global: El body y html se configuran con display: flex, justify-content: center y align-items: flex-start para centrar horizontalmente el contenido principal y alinearlo al inicio verticalmente (evitando que todo el contenido flote en el medio de la pantalla si hay poco contenido).
o	.video-selector-container:
	text-align: center; centra el texto y los elementos inline-block dentro de él.
	padding, margin-top, background-color, border-radius, y box-shadow le dan el aspecto de tarjeta flotante y moderno.
	max-width y width aseguran que sea responsivo y no se estire demasiado.
o	h1: Estilos para el tamaño y color del título.
o	.select-wrapper:
	position: relative; es fundamental para posicionar la flecha ::after en relación con este contenedor.
	display: inline-block; y width: 100%; permiten que el select se expanda completamente dentro de su contenedor padre centrado.
o	#videoSelect:
	width: 100%;, padding, font-size: Estilos básicos para el tamaño y el espaciado.
	border, border-radius: Borde y esquinas redondeadas.
	background-color, cursor: pointer, outline: none: Mejoras de usabilidad.
	-webkit-appearance: none; -moz-appearance: none; appearance: none;: Esto es clave para eliminar la flecha por defecto del navegador, lo que nos permite usar nuestra propia flecha personalizada.
	transition: Suaviza el cambio de color del borde y la sombra al enfocar.
	&:focus: Estilos aplicados cuando el select tiene el foco, mejorando la interacción.
o	.select-wrapper::after (La flecha personalizada):
	content: '⌄';: Inserta el carácter Unicode de una flecha hacia abajo.
	position: absolute; right: 15px; top: 50%; transform: translateY(-50%);: Posiciona la flecha en el lado derecho, verticalmente centrada.
	pointer-events: none;: Asegura que el clic en la flecha "pase a través" hacia el select de abajo, permitiendo abrirlo.
